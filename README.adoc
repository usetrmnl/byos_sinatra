== Setup

To set up project, run:

[source,bash]
----
git clone https://github.com/usetrmnl/byos_sinatra
cd byos_sinatra
bin/setup
----

== Usage

== Development

To contribute, run:

[source,bash]
----
git clone https://github.com/usetrmnl/byos_sinatra
cd byos_sinatra
bin/setup
----

You can also use the IRB console for direct access to all objects:

[source,bash]
----
bin/console
----

== Tests

To test, run:

[source,bash]
----
bin/rake
----
